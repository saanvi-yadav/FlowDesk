# FlowDesk

Team management app — React + Flask + MySQL

## Frontend setup
npm install
npm run dev

## Backend setup
pip install flask flask-cors mysql-connector-python werkzeug
python app.py

## Database setup
mysql -u root -p flowdesk < database/flowdesk_dump.sql

## Runs on
Frontend → http://localhost:5173
Backend  → http://127.0.0.1:5000
